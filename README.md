## Installation

To get a local copy up and running follow these simple example steps.

1. Clone the repo
   ```sh
   git clone https://github.com/jempico/meme-generator.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run
     ```sh
   npm start
   ```
5. Visit http://localhost:3000/
